## âœ¨ Key Features

* **Elegant Palette:** A dark color scheme dominated by crimson (`#FF2E5B` / `#FF335F`) and warm undertones that are easy on the eyes.
* **Soft Contrast:** Engineered to reduce eye strain with a background-to-foreground contrast that avoids pure blacks and harsh whites.
* **Readability Focus:** Syntax highlighting uses vibrant colors (blue for functions, yellow for type names, magenta for constants) to enhance code distinction and comprehension.
* **Harmonious Terminal:** The terminal uses an ANSI color palette that integrates seamlessly with the rest of the editor's aesthetic.
